RCS tasks in short:

  1. Receive alert mail and create ticket
  2. Troubleshoot connectivity
  3. If unsuccessful troubleshooting, create 2 sub tickets to MSS BO: One for SIM activation and one for evaluating if the customer should be shared for the activation.
  4. When resolved notification is received created sub ticket to MSS BO for deactivation of SIM.

 

The following alert mails are sent to servicedesk@marlink.com in case of outage:

 

  1. After 1 hour down FIREING notification will come,
  2. There is a RESOLVED notification sent right after the issue is confirmed resolved. (The criteria is for one bearer to be online again -+9 min because it checks status every 10min).
